## Processing Protocol

**When to process:** Weekly (Sunday) or when items > 20

**How to process each item:**

1. **Is this actionable?**
   - No → Delete or move to "Someday/Maybe"
   - Yes → Continue

2. **Does this belong to a project?**
   - Yes → Move to project file
   - No → Continue

3. **Is this a single task or multi-step?**
   - Single → Add to Tasks-Backlog
   - Multi-step → Create project file

4. **Is this time-sensitive?**
   - Yes → Add due date, schedule
   - No → Backlog with priority
